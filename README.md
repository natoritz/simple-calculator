# Ubuntu/Debian
1. Buka terminal
2. Jalankan perintah berikut:
- sudo apt-get update
- sudo apt-get install python3-tk

# Fedora
1. Buka terminal
2. Jalankan perintah berikut:
- sudo dnf install python3-tkinter

# MacOs
1. Buka terminal.
2. Jalankan perintah berikut:
- brew install python-tk

Jika Anda tidak memiliki Homebrew, Anda dapat menginstalnya di brew.sh.
